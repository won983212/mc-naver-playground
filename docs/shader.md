## 쉐이더 팩 적용 방법

기본으로 쉐이더 팩 3개 설치되어 있습니다. 원하시는 다른 쉐이더 팩이 있으시다면 수동으로 설치해주세요.

1. `설정 > 비디오 설정 > 쉐이더팩 설정`으로 이동한다.
2. 사용하고자 하는 쉐이더 팩을 선택한다.
3. 아래 적용 버튼을 누른다.

## 수동으로 쉐이더 팩 설치하는 방법

쉐이더 팩은 zip 파일 형식으로 되어있습니다.

1. [마인크래프트가 설치된 폴더](index.md)로 이동한다.
2. shaderpacks 폴더에 쉐이더 팩(zip)을 넣는다.
    - 폴더가 없다면 생성해주세요.
